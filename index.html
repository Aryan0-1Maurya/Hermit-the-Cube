Aim of the Game

The Game is Level based and in every Level you need to fill every path field with the yellow color with moving the cube into the directions!

Move on PC

To move on the PC with keyboard you use the Buttons W,A,S,D!

Move on Moblie Devices

To move on Moblie Devices just Swipe into the wanted direction!

How to Editor

First of all give an input into the Input Field at the top(Notice: if the Number is too big the created table will look weird).
Next Click on "Go!". The created table has several Checkboxes(looking like fields) which you can tap on to create the path of your Level!
If you are ready with your level click on "Play!" to create it!
